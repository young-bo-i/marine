# Skills（品牌话术包）

每个品牌一个文件夹 `skills/<品牌>/`。**挂载** = 把文件夹放进来；**移除** = 删掉文件夹。
Scholay 当前使用五类文件：

| 文件 | 对应你说的 | 作用 |
|---|---|---|
| `品牌.md` | 品牌化的内容 | 品牌是什么、可植入的卖点、链接、红线 |
| `执行口径.md` | 生成边界 | 母稿地位、亲历叙事、直评/回复约束 |
| `母稿.md` | 原版话术 | 唯一权威语感，六段正文逐字保留 |
| `母稿索引.json` | 场景路由 | 按页面内容最多选择两段最相关母稿 |
| `风格参数.json` | 风格化的参数设定 | 毒舌度、长度、平台、CTA 强度、直评/回复条数等可调旋钮 |

`评论口径.md` 和 `范文.md` 是旧版参考资料，运行时不再整份加载，避免旧改写和大量样例稀释母稿语感。

## 新增一个品牌
复制 `_模板/` → 改名为品牌名（如 `skills/你的品牌/`）→ 按各文件里的提示填写。

## AI 连接器怎么用
总流程与输出契约见项目根目录的 `AGENTS.md`。插件会先按「字幕 → 评论 → 结构化正文」选择唯一页面来源，再根据内容路由母稿，把精简后的话术包交给 RimeBuffer 当前选中的 AI 连接器。
