# Skills（品牌话术包）

每个品牌一个文件夹 `skills/<品牌>/`。**挂载** = 把文件夹放进来；**移除** = 删掉文件夹。
每个品牌包含三类文件：

| 文件 | 对应你说的 | 作用 |
|---|---|---|
| `品牌.md` | 品牌化的内容 | 品牌是什么、可植入的卖点、链接、红线 |
| `评论口径.md` | 评论口径 | 调性原则、直评/回复策略、Do/Don't、Few-shot 样例 |
| `风格参数.json` | 风格化的参数设定 | 毒舌度、长度、平台、CTA 强度、直评/回复条数等可调旋钮 |

## 新增一个品牌
复制 `_模板/` → 改名为品牌名（如 `skills/你的品牌/`）→ 按各文件里的提示填写。

## Codex 怎么用
总流程与输出契约见项目根目录的 `AGENTS.md`。插件「本地智能体」按钮会把抓取内容交给 Codex、定位到本项目、并让它读对应品牌的三个文件，返回「直评 + 回复」。
